Build all three projects in the Solution.

Port for SQL database is set to 3306
user:- root
password :- root

the Schema that is being used is "project2"

To interact with the Chat client following are the Commands:-

REGISTER <email> <password> - to Signup to the authentication service
AUTHENTICATE <email> <password> - to Login to the authentication service
CONNECT - to Connect to the Chat server
DISCONNECT - to Disconnectg chat server
JOIN <room> - to Join a room.
LEAVE <room> - to Leave a room.
SEND <room> <message> - to send a message to a room.
